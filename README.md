# trading-analytics-dashboard
Project Title:
Real-Time Trading Analytics Dashboard using WebSockets

Description:
An end-to-end analytical application that ingests live Binance market data, processes it in real time, computes key quantitative analytics, and visualizes insights through an interactive dashboard.

Features:
Live tick data ingestion, real-time price visualization, z-score based analytics, alert generation, symbol selection, and CSV export of processed data.

Technologies Used:
Python (FastAPI, Streamlit), WebSockets, SQLite, Pandas, Plotly, HTML, JavaScript, and Binance Futures WebSocket API.
